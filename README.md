Rademade Catalog
=================

Simple catalog of items

* Ruby version 2.5.1
* Rails version 5.2.1


System dependencies
-------------------
You should have the installed ruby version 2.5.1 and PostgreSQL.

Database setup
-----------------
Simply run:

    $ bin/setup

Tests
-----
To run the test suite just run:

    $ bundle exec rspec

Run instructions
----------------
Once you have set up your environment, you could run your application.
##### Production:

    # bundle exec rails s
